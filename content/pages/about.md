# About This Blog

This blog is built with [Zola](https://www.getzola.org/), a blazingly fast static site generator written in Rust, and hosted on [GitHub Pages](https://pages.github.com/). It uses the [tabi](https://github.com/welpo/tabi) theme for a modern, clean look with perfect Lighthouse scores and built-in dark mode support.

## Technical Details

- **Static Site Generator**: Zola 0.19+
- **Theme**: tabi (modern, accessible, performant)
- **Hosting**: GitHub Pages
- **Deployment**: GitHub Actions (automatic on push)
- **Development Environment**: Nix Shell with Zola
- **Build Time**: ~0.13 seconds (22x faster than Jekyll!)

## Why Zola?

After using Jekyll, I migrated to Zola for several compelling reasons:

### Performance
- **Jekyll**: 2.9+ seconds per build
- **Zola**: 0.13 seconds per build
- **22x faster** - makes development a breeze!

### Simplicity
- **No Dependencies**: Single binary, no Ruby/Bundler/gems
- **Everything Built-in**: Sass, syntax highlighting, RSS, search, sitemap
- **Zero Configuration**: Sensible defaults that just work

### Developer Experience
- **Instant Rebuilds**: See changes immediately
- **Clear Error Messages**: Know exactly what's wrong
- **Modern Stack**: Rust-based, maintained, actively developed
